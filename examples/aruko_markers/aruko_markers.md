# Aruko Markers Examples

This folder contains examples for chaining controller functionalities to detect and react to Aruko markers.
These examples are more complex than the others.


## What's Here

- [find_markers_sim.py](find_markers_sim.py) — Program showcasing Aruko detection and reaction in the Mujoco Simulator
- [find_markers_native.py](find_markers_native.py) — Program showcasing Aruko detection and reaction on the native Go2 robot.
- [aruko_helpers.py](aruko_helpers.py) — Aruko marker detection helpers used by the examples.
